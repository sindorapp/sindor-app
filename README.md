# SINDOR — Avaliação Continuada de Dor Crônica

Plataforma clínica segura para acompanhamento longitudinal de pacientes com dor crônica, baseada em instrumentos validados internacionalmente.

## 📋 Sobre

SINDOR é um aplicativo web desenvolvido pelo **Dr. Leonardo Botelho** no **Serviço Interdisciplinar de Dor** em Porto Alegre, RS. A plataforma implementa um sistema completo de avaliação de dor crônica com múltiplos questionários validados e acompanhamento de progresso do paciente.

## ✨ Recursos Principais

### Para Pacientes
- 📊 **Avaliações Longitudinais** - Responda questionários padronizados internacionalmente
- 📈 **Gráficos de Progresso** - Visualize sua evolução ao longo do tempo
- 🔒 **Controle de Privacidade (LGPD)** - Autorize quais profissionais acessam seus dados
- 💾 **Salvar Progresso** - Interrompa e continue sua avaliação quando desejar
- 📱 **Interface Responsiva** - Funciona em desktop, tablet e mobile

### Para Médicos
- 👥 **Lista de Pacientes** - Gerenciamento centralizado de pacientes
- 📊 **Dashboard Individual** - Visualize scores e evolução de cada paciente
- 📝 **Conduta Médica** - Registre medicações, intervenções e notas clínicas
- 📋 **Histórico de Sessões** - Acompanhe todas as avaliações do paciente
- 🔐 **Gestão de Consentimentos** - Controle acesso aos dados dos pacientes

## 📋 Questionários Implementados

1. **BPI** (Brief Pain Inventory) - Intensidade da dor e impacto nas atividades
2. **DASS-21** - Depressão, Ansiedade e Estresse
3. **EPD** (Escala de Pensamentos sobre a Dor) - Catastrofização da dor
4. **PSEQ** - Autoeficácia para Dor
5. **CSI** (Central Sensitization Inventory) - Sensibilização central
6. **SF-36** - Qualidade de Vida Relacionada à Saúde
7. **PSQI** (Pittsburgh Sleep Quality Index) - Qualidade do Sono
8. **EQ-5D** - Estado de Saúde Geral
9. **PGI-I** - Impressão Global de Melhora pelo Paciente

## 🛠️ Tecnologia

- **Frontend**: HTML5, CSS3, JavaScript Vanilla
- **Gráficos**: Chart.js
- **Fonts**: Google Fonts (Playfair Display, DM Sans, DM Mono)
- **Armazenamento**: LocalStorage
- **Tema**: Dark mode com variáveis CSS personalizáveis
- **Conformidade**: LGPD (Lei nº 13.709/2018)

- ✅ Perguntas de segurança para recuperação de senha

## 📖 Como Usar

### Para Pacientes
1. Clique em "Sou Paciente" na tela inicial
2. Crie uma conta com email e senha
3. Autorize os profissionais que poderão acessar seus dados
4. Responda o questionário completo
5. Acompanhe sua evolução na seção "Meu Progresso"

### Para Médicos
1. Clique em "Acesso do Médico" na tela inicial
2. Crie uma conta com CRM válido
3. Acesse a lista de seus pacientes no dashboard
4. Visualize avaliações e registre conduta médica
